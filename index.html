How an AI agent builds and publishes a website

1. Research & structure

Super browses competitor sites, docs, and inspiration sources to outline pages, copy, and structure — the same way a human would.

2. Generate real files

HTML, CSS, and JavaScript are generated as real files. Super can follow patterns similar to tools like Atoms or Genspark, but with full computer control.

3. Push to GitHub

The agent logs into GitHub in a cloud browser, creates a repo, commits files, and opens pull requests — assisted, with user confirmation.

4. Deploy & iterate

Super deploys to hosting, previews changes, and replays the same steps later using the computer-use cache so iteration doesn’t restart from scratch.
